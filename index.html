VARIABLES NAME :- LEGAL AND ILLEGAL

  1. A heading stating “Rules for naming JS variables”
  2. Variable names can only contain & sign,_ sign, numbers and letter.
    For example $my_1stVariable
  3. Variables must begin with a Name, = sign or "" sign. example $name, _name or name
  4. Variable names are case Camel Case
  5. Variable names should not be JS Keywords
